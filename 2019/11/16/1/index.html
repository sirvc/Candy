又一轮‘九四’监管风来袭

发表于 2019-11-16 | 分类于 币圈 |

一份落款为人民银行上海总部互金整治办和市金融稳定联席办的名为《关于开展虚拟货币交易所排摸整治通知》的文件于15日流传于各大微信群和朋友圈。

《通知》内容指出，近期，借区块链技术的推广宣传，虚拟货币炒作有抬头迹象，为防范死灰复燃，根据国家互金整治办相关部署，请各区整治办对辖内以下虚拟货币相关活动进行摸排：

一是在境内组织虚拟货币交易；

二是以“区块链应用场景落地“等为由，发行”xx币”、“xx链”等形式的虚拟货币，募集资金或比特币、以太坊等虚拟货币；

三是注册在境外的ICO项目、虚拟货币交易平台等提供宣传、引流、代理买卖等服务。

《通知》要求在11月22日前完成摸排工作，“一旦发现从事上述虚拟货币相关活动的互联网企业，立即报送市金融稳定联席办（市地方金融监管局）和人民银行上海总部，并督促企业立即整改退出，打早打小。”

自从上个月国家把区块链作为核心技术自主创新重要突破口后，整个区块链行业集体嗨了，包括传统的互联网上市公司也纷纷把自己的业务与区块链概念挂钩，希望股民能够买账。区块链大火，自然少不了骗子，各种空气币纷纷起死回生，打着国产第一公链的名义行骗。更有甚者，打着国家官方数字货币DCEP的名号公然行骗。这个时候泼点冷水也许是好事，稳中求发展。

铁打的比特币，流水的监管。比特币正是在不断的打压中建立起来的强大共识价值，如《通知》所言打早打小，比特币经过10年的发展早已经不再是阿斗，其背后的区块链技术也在比特币的传播下得到了很多人的认可。这次打击的重点其实是各种虚假项目发行的众筹币，还有一些配合项目方割韭菜的十八线交易所。

共识不是涨出来的，是跌出来的。从币价反应上来看，比特币的共识都是最强的。利好放出时，比特币不一定是涨的最多的，利空放出时，比特币一定是跌的最少的。现在的币圈基本都是留存资金在博弈，如果没有新韭菜入场，大概率是一种阴跌的走势，大家都在靠着明年的减半行情维持着信仰。

做个简单的判断，利空出尽是利好，利好出尽是利空。如果比特币从现在到明年减半时是阴跌的走势，那么减半的时间点应该是很好的买入点，因为减半后是抛压减半的实质利好。如果比特币从现在到明年减半时是步步高的走势，那么减半的时间点可能是阶段高点，因为减半前抛压减半的预期利好已经实现。比特币现价8470$。
